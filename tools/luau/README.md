## Vendored Roblox Type Definitions

`globalTypes.None.d.luau` is a vendored third-party analyzer input from the official `luau-lsp` project.

- Source: `https://luau-lsp.pages.dev/type-definitions/globalTypes.None.d.luau`
- Upstream project: `https://github.com/JohnnyMorganz/luau-lsp`
- Purpose: provide Roblox game-script API types to standalone `luau-lsp analyze` runs used by `make typecheck` and `make typecheck-report`

Keep this file pinned and committed as-is. Update it only by replacing it with the upstream published artifact for the intended `luau-lsp` version.
